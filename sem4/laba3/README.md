# Moorhuhn - Игра-тир с курицами

## Общее описание программы
Программа представляет собой аркадную игру-тир с элементами симулятора охоты, реализованную на библиотеке Pygame. Игроку необходимо за 60 секунд настрелять как можно больше кур, которые летают на разных расстояниях. Особенностью игры является система параллакс-фона с препятствиями (деревьями), а также система рекордов.

## Игровая логика

### Класс Chicken (entities.py):
- Хранит уровень курицы (near/mid/far), очки, скорость, масштаб
- Анимация полёта (2 кадра с взмахами крыльев)
- Метод `hit()` — переход в состояние падения
- Поддерживает автоматическое движение слева направо или справа налево
- При попадании курица падает вниз с ускорением (гравитация)

### Класс Crosshair:
- Следует за курсором мыши
- Скрывает системный курсор

### Класс Game (game.py):
- Управляет игровым процессом
- Обрабатывает выстрелы, попадания, перезарядку
- Отслеживает время (60 секунд)
- Проверяет попадание в препятствия (деревья)
- Обрабатывает окончание времени и рекорды

### Система очков:

| Расстояние | Цвет | Очки |
|------------|------|------|
| Близкое (near) | Красный | 10 |
| Среднее (mid) | Зеленый | 20 |
| Далёкое (far) | Синий | 30 |

## Параллакс-фон и препятствия

### Класс ParallaxBackground (parallax.py):
- Реализует многослойный параллакс с 4 слоями:
  - Небо (фиксированное)
  - Облака (скорость 0.3)
  - Поле (скорость 0.8) 
  - Деревья (скорость 1.4)
- **Принцип работы**: Слои двигаются с разной скоростью при движении мыши, создавая эффект глубины
- Деревья выступают в роли препятствий — пули застревают в них
- Используется маска для определения попадания в деревья
- Следы от пуль на деревьях визуализируются с эффектом затухания

### Визуальные эффекты:
- **Параллакс**: Многослойная прокрутка с разной скоростью
- **Следы от пуль**: Отображение отверстий на деревьях с прозрачностью
- **Анимация падения**: Курицы падают с ускорением

## Система рекордов

### Класс RecordManager:
- Хранит топ-3 рекорда в файле `records.json`
- Проверяет, попадает ли результат в таблицу рекордов
- Сохраняет имя игрока и количество очков
- Поддерживает удаление всех рекордов

## Игровой процесс

### Начало игры:
1. Игрок выбирает пункт "Начать игру" в главном меню
2. Игра начинается с 60 секундами и 8 патронами
3. Курицы появляются с левого или правого края экрана

### Механика стрельбы:
- При клике ЛКМ проверяется попадание в прицел
- Проверка на попадание в деревья (с использованием маски)
- Если попадание в курицу: начисляются очки, курица падает
- Если попадание в дерево: появляется след от пули

### Окончание игры:
- При окончании времени игра останавливается
- Если результат входит в топ-3 — запрос имени для рекорда
- Отображение таблицы рекордов

## Анимации в проекте

| Название анимации | Тип по способу реализации | Описание |
|-------------------|---------------------------|----------|
| Анимация полёта курицы | Кадровая анимация | 2 кадра с взмахами крыльев, смена с интервалом в 10 кадров |
| Анимация падения курицы | Физическая | Курица падает с ускорением (гравитация 0.5) после попадания |
| Анимация параллакса | Физическая | Слои фона двигаются с разной скоростью при движении мыши |
| Следы от пуль | Трансформационная | Плавное затухание прозрачности (альфа) с течением времени |
| Мигание времени | Анимация состояния | При остатке времени менее 10 секунд — мигание красным/оранжевым |
| Подсветка выбранного пункта меню | Анимация состояния | Жёлтая подсветка текущего выбранного пункта меню |
| Анимация имени при рекорде | Интерфейсная | Поле ввода с отображением вводимого имени |
